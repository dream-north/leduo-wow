# 乐多汪汪 (Leduo Wow)

macOS 语音输入助手，基于 Qwen ASR 和 LLM 构建。

## 功能特性

- **语音识别** - 按一下快捷键说话，再按一下后自动识别并输入文字
- **语音助手** - 语音提问，AI 智能回答
- **文本润色** - 可选 LLM 润色功能，优化识别结果
- **双模式支持** - 支持左右修饰键区分（如 RightCommand / RightOption）
- **全局快捷键** - 系统级快捷键，在任何应用中都可触发

## 系统要求

- macOS 13.0 或更高版本
- 需要授予辅助功能权限（用于全局快捷键监听）
- 需要授予麦克风权限
